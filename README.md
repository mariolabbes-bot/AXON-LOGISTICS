# AXON LOGISTICS - Plataformas de Transporte Integradas

Sistema independiente de gestión de transporte (TMS) integrado con AXON CRM.

## Arquitectura
- **Frontend**: Next.js 15 (Vercel)
- **Backend**: Express / Node.js (Render)
- **Database**: PostgreSQL (Neon)

## Despliegue
Este proyecto está configurado como un Monorepo de npm Workspaces.

### Frontend
- Ubicación: `/frontend`
- Configurado con Modo de Alto Contraste para operaciones de terreno.

### Backend
- Ubicación: `/backend`
- API REST para gestión de facturas y enrolamiento de productos.
